2.Amazon UI Clone

I developed a fully responsive UI clone of the Amazon homepage using HTML and CSS, replicating the core design, layout, and structure of the original website. This project aimed to enhance my frontend development skills, focusing on UI consistency, styling, and layout structuring.

Key Features:

  • Fully structured homepage layout – Includes the header, navigation bar, product sections, and footer to replicate the real Amazon experience.
  • Header and Footer – Implemented Amazon's search bar, cart icon, and category navigation menu for an authentic feel.
  • Product Display Sections – Added sample product listings with hover effects to improve user interaction.

This project helped me understand modern UI trends, layout structuring, and responsive web design techniques.

2.Diabetic Foot Ulcer Detection

This project utilizes Machine Learning techniques to detect and classify diabetic foot ulcers into different severity stages. Early detection can significantly improve treatment outcomes for patients.

Our approach involved:

  • Using deep learning models for image classification
  • Training the model on a dataset of medical images
  • Implementing an interactive user interface for medical professionals

The project was successfully validated and published in a research paper, demonstrating its potential impact in the medical field.
